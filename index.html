Regras do Jogo

O que vai te derrubar
e como evitar

ARMADILHA 01
Tutorial Hell
Você vai querer assistir curso de JavaScript, Python, Node antes de começar. Não faça isso. O Claude Code explica enquanto você constrói. Tutorial é procrastinação disfarçada de produtividade.
ARMADILHA 02
Projeto Perfeito
Você tem instinto de designer — vai querer que o código fique bonito e organizado. Semanas 1 e 2: não importa. Feio e funcionando vale mais que bonito e travado. Refatora depois.
ARMADILHA 03
Pular Etapas
É tentador pular a Semana 1 porque parece simples. Não pule. Um erro que você não entende no terminal vai te travar por dias. Os fundamentos pagam dividendos infinitos.
COMBUSTÍVEL 01
1 hora todo dia
Consistência vence intensidade. Commit diário no GitHub — seu histórico de progresso visível. Fale com seu amigo semanalmente. Ele já passou por isso e pode te salvar horas.
COMBUSTÍVEL 02
Publica o que aprende
Post semanal no X sobre o que construiu. Aprende quem ensina. Você vai construir autoridade enquanto aprende — não depois. Esse é o meta-produto do plano.
COMBUSTÍVEL 03
O prompt certo
Errado: "Como faço um loop em JavaScript?" — Certo: "Preciso ler cada linha de um CSV e gerar um texto para cada uma. Como faço isso nesse arquivo que já tenho?" Contexto + objetivo = respostas 10x melhores.
